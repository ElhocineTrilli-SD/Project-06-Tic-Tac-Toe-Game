# Tic Tac Toe Game (Desktop App)

A simple **Tic Tac Toe** desktop game built using **C#** and **Windows Forms**. This application allows two players to play Tic Tac Toe on a single computer with a visual interface.

## 📝 Features

- Two-player gameplay (Player 1 and Player 2)
- Interactive buttons for each cell
- Player turns indicated on the interface
- Win detection for rows, columns, and diagonals
- Draw detection
- Game restart functionality
- Visual feedback for winning cells

## 💻 Technology Used

- **Language:** C#
- **Framework:** .NET Windows Forms
- **IDE:** Visual Studio
- **Optional:** SQL Server (if saving game scores, not implemented in this version)

## 🎮 How to Play

1. Open the project in **Visual Studio**.
2. Build and run the project (`F5` or `Start`).
3. The game window will display a 3x3 grid.
4. Players take turns clicking on empty cells:
    - Player 1 uses **X**
    - Player 2 uses **O**
5. The game detects the winner or draw automatically.
6. Press the **Restart Game** button to play again.
